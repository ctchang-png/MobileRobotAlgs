This is the student distribution for the courseware for 16-362 
Mobile Robot Programming at CMU-RI.

Files and Directories:
----------------------

custom_robot_msgs defines custom ROS messages that are not included in the MATLAB ROS toolkit

dataStructures generic lists, queues etc.

examples are short example code that illustrates use of the distribution

graphics code that produces the display window

Raspbot.m implements the main api to the robot

models - of robots, pallets etc.

mralCodes - code speciically provided for assignemnts

shapes - code for drawing and manipulating objects

sim - the simulator

utils - nore useful more generic robot stuff

raspbot, realRobotIF, robotBaseClass, robotIF, simRobotIF are your interface to the robot and simulator
