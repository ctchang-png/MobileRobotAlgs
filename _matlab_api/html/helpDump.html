helpDump
 helpDump Command for dumping MATLAB .m file markup as help html files.
 
    This class will process .m file markup and produce the standard
    MATLAB html help files. The class is based on the undocumented and
    unsupported MATLAB function help2html, so it may break eventually
    if that function disappears from MATLAB.
 
    You can call this as a function with helpDump(arg1,arg2,...) or as
    a shell command with helpDump arg1 arg2. Arguments do not have to
    be quoted 'thus'.
 
    File descriptors containing the standard wildcard '*' will be
    expanded into a list of files as you might expect. Files missing a
    .m extension will have one added.
 
    Usage:
 
    helpDump file1 file2.m dumps all help markup for file1.m and file2.m
    helpDump *.m dumps all help markup in this directory here.
    helpDump ../*.m dumps all help markup in parent directory here.
 
    The help files appear in the current working directory
